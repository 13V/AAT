# Adelaide Affordable Trailers - Premium Website

A bold, conversion-focused website for Adelaide Affordable Trailers featuring neon yellow branding, interactive pricing calculator, and mobile-first design.

## Features

- **Bold Neon Yellow Branding**: Eye-catching #D4FF00 color scheme on dark background
- **Interactive Pricing Calculator**: Real-time quote estimation for all services

- **Service Showcase**: Detailed presentation of all waste removal services
- **Transparent Pricing**: Clear pricing tables with no hidden fees
- **Contact Forms**: Easy quote request and booking system
- **Responsive Layout**: Perfect display on all devices

## Local Development

Simply open `index.html` in a modern web browser to view the website.

For a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```

Then navigate to `http://localhost:8000`

## Deployment

This is a static website and can be deployed to:
- **Vercel**: Connect your GitHub repository
- **Netlify**: Drag and drop the folder
- **GitHub Pages**: Enable in repository settings
- **Any web hosting**: Upload files via FTP

## Customization

### Colors
Edit CSS custom properties in `styles.css`:
```css
:root {
    --primary: #D4FF00;
    --dark: #1a1a1a;
}
```

### Content
- Update phone number (0422 529 662) throughout the site
- Add actual business address and ABN
- Update service areas
- Add real customer testimonials
- Update pricing if needed

### Images
